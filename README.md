### Ideias de projetos
 - Engine feita em C++ e SDL2 com implementação de cálculos físicos, matrizes de rotação e, para uma segunda fase, projeção;
 - Algoritmo para criar fórmulas da Lógica Proposicional, criar árvores, avaliá-las e escrevê-las em pré-ordem, em-ordem e pós-ordem (A princípio, em C++);
 - Algoritmos para operar matrizes, calcular determinantes e manipular filas;
 - Utilizar Codon para compilar um arquivo-exemplo python;
 - Script para formatar um arquivo de texto em sua respectiva linguagem;
 - Refatorar projeto py-hamming.
 
 Há esboços no repositório [ideias-nao-requintadas](https://github.com/mathsmm/ideias-nao-requintadas).
